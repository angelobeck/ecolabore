Ao criar uma senha, recomendamos o seguinte:

  • Ter pelo menos 8 caracteres
  • Conter letras maiúsculas, minúsculas, números e símbolos
  • Não incluir caractéres acentuados ou outros diacríticos
  • Não utilizar seus dados pessoais como nome ou data de nascimento para compor a senha

As senhas não são iguais
