# ecolabore

secure, extensive and accessible CMS

Main features:

* All languages
  - Manage your pages in many languages you need
  - System tools in portuguese and english. Please help us to translate to other languages!
* Secure
  - Database encryption
  - Server files protection
  - Data filtering and many types of attacks and injections block
* Accessible
  - Web Content Accessibility Guidelines (WCAG) compliant
  - 100% tested with many screen readers in many browsers
* Simple
  - Display only relevant contents and modules
  - Create only the sections you need
  - Sort the menus as you like
  - Display only the details you whant for each post
* Responsive
  - Mobile first layout system
  - Define modules and images widths for 4 screen sizes
* Flexible
  - Create formularies easily
  - Custom colors, fonts, positioning, widths, margins in a friendly interface
  - Edit HTML, CSS and JavaScript freely
* Extensible
  - Install and manage modules
  - Ready to install templates. Help us to develop beauty templates!
* Comunicative
  - Send SMTP and SMS messages to the clients
  - Generate RSS feeds dinamicaly
  - Generate robots.txt and sitemap.xml
  - Configure Google Webmasters site owner verification
* Administrative area
  - Add and remove users
  - Add and remove domains
  - System configuration
  - Database options
  - Components management
  - System alerts, security report and more tools
* User profile area
  - Update your profile information
  - Change password
  - Check mail and phone
  - View user projects and publications and much more
* Create your site
  - Preconfigured blogs for galeries, news, events, publications, podcasts and other needs
  - Create customizable formularies for contact, events subscriptions, ordering, etc
  - Pannels with dinamic contents
  - Format the content with MarkDown
  - Insert images, files for download, audios to listen, cards, grids, columns, tables and more
  - Keywords glossary
  - Customize "404 Not found" page, search results page, terms of service, privacy policy and site informations page

We know that some features are not working, like:

* Shop cart processing
* System auto packager
* Video functionalities
* Financial administration features
* Some images features, like the banner module
* Upload big files

We decide to not remove these features from the master branch, because we know that we will work there in the future.
